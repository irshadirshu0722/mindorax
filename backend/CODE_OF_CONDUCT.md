# Code Of Conduct

## Our Commitment

We want Mindorax Backend to be a welcoming, respectful project for learners, contributors, and maintainers. Everyone participating in this repository is expected to help create a professional and harassment-free environment.

## Expected Behavior

- Be respectful in feedback and review comments.
- Assume good intent and communicate clearly.
- Focus criticism on ideas, code, and behavior, not people.
- Share context generously, especially when reviewing newer contributors.
- Accept constructive feedback gracefully.

## Unacceptable Behavior

- Harassment, insults, or discriminatory language
- Trolling or intentionally disruptive behavior
- Publishing someone else's private information
- Repeatedly derailing technical discussions
- Hostile or exclusionary conduct in issues, pull requests, or chats

## Enforcement

Project maintainers may remove, edit, or reject comments, commits, issues, or pull requests that violate this code of conduct.

## Reporting

If you experience or witness unacceptable behavior, report it privately through the channel described in [`SECURITY.md`](SECURITY.md) or directly to the maintainers if the issue is community-related rather than security-related.
