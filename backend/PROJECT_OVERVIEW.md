# Project Overview

## Small Description

Mindorax Backend is an AI-powered study planner backend built with Django and Django Ninja. It helps students manage subjects, upload study material, generate AI-based study plans and quizzes, and track learning progress through structured APIs.

## Tech Stack

- Python 3.13
- Django 6
- Django Ninja
- PostgreSQL
- Redis
- Celery
- Gunicorn
- Google OAuth
- Gemini AI
- Docker and Docker Compose

## Key Features

- Google-based authentication with JWT token flow
- Subject creation, update, retrieval, and deletion
- Subject file upload and file metadata management
- AI subject analysis with topics, concepts, and study insights
- AI-generated study plans with plan items and study sessions
- AI-generated quizzes with question options and scoring
- Quiz attempt submission and result tracking
- Background task processing with Celery
- Failed task logging for operational visibility
